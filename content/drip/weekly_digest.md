## Authoring Notes

- Only include signals that resolved this week (market closed + outcome confirmed)
- If fewer than 3 resolved: include however many did. Don't pad with unresolved signals.
- If zero resolved: skip the digest for that week — don't send an empty update
- "One thing we learned" should be honest about both correct and incorrect calls
- Never frame a wrong call as a near-miss or hedge it — just say what we got wrong
